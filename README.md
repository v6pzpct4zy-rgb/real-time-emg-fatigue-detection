# Real-Time EMG-Based Muscle Fatigue Detection System

This repository contains the implementation of my graduation project in Electrical and Electronics Engineering.

The project focuses on real-time muscle fatigue detection using surface EMG signals. The system uses a surface EMG sensor, Arduino Nano, and Raspberry Pi 4 to acquire and process EMG signals, and a lightweight 1D-CNN model to classify signal windows into rest, pre-fatigue, and fatigue states.

## Project Highlights

- Graduation project in Electrical and Electronics Engineering
- Real-time surface EMG data acquisition
- Arduino Nano and Raspberry Pi 4 embedded workflow
- UART-based serial communication
- Sliding-window signal processing
- 1D-CNN-based three-class classification
- Approximately 84% classification accuracy
- MATLAB/Python-based signal analysis and visualization

## Data Privacy

The full EMG dataset is not publicly shared due to participant privacy. This repository includes only anonymized sample data, selected scripts, trained model files, and project result figures.
